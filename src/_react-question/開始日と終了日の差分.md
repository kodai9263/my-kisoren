1. やりたいこと
「開始日」と「終了日」を選ぶと、その2つの間の“差分日数（端は含まない）」を表示します。
開始・終了のどちらかが未選択の間は、グレーで「開始日と終了日を選択してください」と表示します。
終了日が開始日より前の場合は、赤字でエラーメッセージを表示します。

2. 実装の条件
コンポーネント名は DateRangeDaysBasic とします。
useState で start: string と end: string（初期値はいずれも ''）を管理し、両方の <input type="date"> を“制御コンポーネント”として実装します。
差分日数は、両方が入力されたときにのみ計算します。
const s = new Date(start); const e = new Date(end);
const msPerDay = 1000 * 60 * 60 * 24;
const days = Math.round((e.getTime() - s.getTime()) / msPerDay);（端は含まない差）

表示仕様：
未選択時: 灰色で「開始日と終了日を選択してください」
エラー時（終了日 < 開始日）: 赤字で「終了日は開始日以降にしてください」
正常時: 緑系で「差分: X日」
